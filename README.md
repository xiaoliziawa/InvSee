# InvSee

A simple item marking mod for Minecraft.

## Features

**Mark Items in Containers**
- Press middle mouse button on any item to create a floating marker
- Shows item name, owner name, and container outline
- Works in any container - chests, machines, backpacks, etc.
- Also works with JEI items

**Slot Highlighting**
- Marked items get highlighted in all open containers
- Helps you quickly find matching items across inventories

**FTB Teams Support**
- Party members can see each other's marks
- Players not in a party can see marks from other solo players
- Party players and solo players cannot see each other's marks
- Without FTB Teams installed, all players see all marks

## Configuration

- Mark duration (1-60 seconds)
- Display range (16-256 blocks)
- Highlight duration
- Toggle tooltip display
- Toggle extra item info (durability, enchantments, energy, etc.)

## Mod Compatibility

- JEI - Mark items directly from recipe viewer
- Mekanism - Shows chemical tank info
- Applied Energistics 2 - Shows storage cell status
- FTB Teams - Team-based mark visibility

## Requirements

- Minecraft 1.21.1
- NeoForge
